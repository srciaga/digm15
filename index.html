DIGM15: Web Design Fundamentals

Fall 2021 Portfolio

Course Description:

This course is an introduction to web design and user experience. Fundamentals of UX and visual interface design, interaction design, and GUI development will be introduced.
Students will learn the basics of HTML and CSS. Additionally, students will learn to create and optimize raster and vector images, and embed existing video for the Web.
